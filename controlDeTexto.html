<img>: el elemento de inserción de imagen El elemento HTML <img> incrusta una imagen en el documento. Se usa para colocar imagenes,de diferentes tipos ya sean jpg , gifs, etc...
NOTA: Mejor utilizar svg que jpg para no perder calidad
<tabla>: El elemento Tabla El elemento HTML <tabla> representa datos tabulares, es decir, información presentada en una tabla bidimensional compuesta por filas y columnas de celdas que contienen datos.

<tbody>: el elemento del cuerpo de la tabla El elemento HTML <tbody> encapsula un conjunto de filas de tabla (elementos <tr>), lo que indica que comprenden el cuerpo de la tabla (<table>). El elemento <tbody>, junto con sus elementos <thead> y <tfoot> relacionados, proporcionan información semántica útil que se puede utilizar al renderizar para pantalla o impresora.
<td>: El elemento Celda de datos de la tabla El elemento HTML <td> define una celda de una tabla que contiene datos. Participa en el modelo de tabla. Se puede utilizar para hacer tablas.
<tfoot>: El elemento Pie de tabla El elemento HTML <tfoot> define un conjunto de filas que resumen las columnas de la tabla. Se puede utilizar para hacer tablas.
<th>: el elemento de encabezado de tabla El elemento HTML <th> define una celda como encabezado de un grupo de celdas de tabla. La naturaleza exacta de este grupo está definida por los atributos de alcance y encabezados. Se puede utilizar para hacer tablas.
<thead>: el elemento de encabezado de tabla El elemento HTML <thead> define un conjunto de filas que definen el encabezado de las columnas de la tabla. Se puede utilizar para hacer tablas
<tr>: El elemento Fila de la tabla El elemento HTML <tr> define una fila de celdas en una tabla Las celdas de la fila se pueden establecer usando una combinación de elementos <td> (celda de datos) y <th> (celda de encabezado)
<a>: El elemento ancla El elemento HTML <a> (o elemento ancla), con su atributo href, crea un hipervínculo a páginas web, archivos, direcciones de correo electrónico, ubicaciones en la misma página o cualquier otra cosa que una URL pueda abordar. El contenido dentro de cada <a> debe indicar el destino del enlace. Si el atributo href está presente, presionar la tecla Intro mientras se enfoca en el elemento <a> lo activará.
